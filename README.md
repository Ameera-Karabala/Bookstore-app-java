# Bookstore-app-java
Java Swing bookstore application 

This is a bookstore application I built as a personal project. It uses Java Swing for the GUI and follows object-oriented design.

The app has two types of users: owner and customer.

The owner can:
- add and delete books
- add and delete customers

Customers can:
- log in
- view available books
- buy books
- redeem points

The system keeps track of points and assigns a status (Silver or Gold) based on how many points the customer has.

This project helped me practice working with GUI (Swing), multiple classes, and design patterns like the state pattern.

To run the program, run MainFrame.java.
